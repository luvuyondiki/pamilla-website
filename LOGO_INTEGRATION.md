# UBUNTU VOYAGE Logo Integration

## Logo Files Created

1. **`logo.svg`** - Main logo for navigation and footer
2. **`logo-simple.svg`** - Simplified version for mobile/smaller spaces
3. **`favicon.svg`** - Favicon for browser tabs

## Logo Placement

### 1. Navigation Bar
- **Location**: Top navigation bar
- **File**: `logo.svg`
- **Size**: 44px x 44px (responsive)
- **Styling**: `.logo-image` class

### 2. Hero Section
- **Location**: Main hero section above the title
- **File**: `logo.svg`
- **Size**: 200px width (responsive: 150px on tablet, 120px on mobile)
- **Styling**: `.hero-logo-image` class with hover effects

### 3. Footer
- **Location**: Footer section
- **File**: `logo.svg`
- **Size**: 60px x 60px
- **Styling**: `.footer-logo-image` class

### 4. Browser Tab (Favicon)
- **Location**: Browser tab icon
- **File**: `favicon.svg`
- **Size**: 32px x 32px

## Logo Design Elements

- **Globe**: Circular outline with continent shapes
- **Location Pin**: Positioned on the globe with spiral design
- **Text**: "UBUNTU VOYAGE" in bold
- **Tagline**: "RECRUITMENT & TRAVEL AGENCY" in lighter weight
- **Colors**: Dark teal (#1e3a8a) and light grey (#64748b)

## Responsive Behavior

- **Desktop**: Full logo with text
- **Tablet**: Slightly smaller logo
- **Mobile**: Compact logo with responsive sizing

## CSS Classes Used

- `.logo-image` - Navigation logo
- `.hero-logo-image` - Hero section logo
- `.footer-logo-image` - Footer logo

## Browser Support

- SVG format ensures crisp display at all sizes
- Fallback to PNG if needed (not implemented)
- Modern browsers support SVG favicons
